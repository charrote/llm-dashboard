什么是 OpenCode？

OpenCode 是一款 AI 代码编辑器，提供智能代码补全、代码解释、代码生成等功能。

第一步：下载 OpenCode

1. 访问 OpenCode 官网或 GitHub 下载最新版本

2. 安装并启动 OpenCode

第二步：获取 APIKey

在查询页面获取您的专属 APIKey：

→ 前往查询 APIKey

第三步：配置 OpenCode

1. 打开设置界面（Settings）

2. 找到 AI / LLM 设置项

3. 配置 API Endpoint 和 API Key

// 配置示例

// API Endpoint:

http://nat.ywapi.com:9234/v1

// API Key:

第四步：开始使用

配置完成后，即可开始使用 AI 辅助编程。

  • 输入代码时自动获取补全建议
  • 使用快捷键触发 AI 助手
  • 支持代码解释和生成

常见问题

Q: 连接失败怎么办？

A: 检查 API Endpoint 是否正确，确保服务器地址可访问。

Q: 没有补全提示？

A: 确认 APIKey 正确且未过期，检查网络连接。

Q: 其他相似工具？

A: Roo、Cline、Continue 等使用方法一样，配置相同的 API Endpoint 和 API Key 即可。
