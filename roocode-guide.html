什么是 Roocode？

Roocode 是 VSCode 的 AI 代码补全插件，可以帮助你快速编写代码、提供智能提示和建议。

第一步：安装 Roocode 插件

1. 打开 VSCode

2. 点击左侧扩展图标（或按 Ctrl+Shift+X）

3. 搜索 "Roocode"

4. 点击安装

第二步：获取 APIKey

在查询页面获取您的专属 APIKey：

→ 前往查询 APIKey

第三步：配置 Roocode

1. 点击 VSCode 左下角设置图标，选择 Settings

2. 搜索 "Roocode" 或找到 Roocode 设置项

3. 找到 API Endpoint 和 API Key 配置项

// 配置示例

// API Endpoint:

http://nat.ywapi.com:9234/v1

// API Key:

ux-XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

第四步：开始使用

配置完成后，重启 VSCode 即可开始使用。

  • 开始输入代码时，Roocode 会自动提供补全建议
  • 按 Tab 接受建议
  • 按 Ctrl+↓ 查看更多建议
  • 按 Ctrl+Shift+P 输入 "Roocode" 查看更多命令

常见问题

Q: 连接失败怎么办？

A: 检查 API Endpoint 是否正确，确保服务器地址可访问。

Q: 没有补全提示？

A: 确认 APIKey 正确且未过期，检查网络连接。

Q: 其他相似工具？

A: Roo、Cline、Continue 等使用方法一样，配置相同的 API Endpoint 和 API Key 即可。
