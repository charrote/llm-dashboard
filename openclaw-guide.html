什么是 OpenCLAW？

OpenCLAW 是一个开源的 AI 编程助手，通过命令行与用户交互。它运行在本地的 AI 网关上，支持连接多种 AI 提供商。

第一步：安装 OpenCLAW

1. 确保已安装 Node.js 22 或更高版本

2. 在终端执行以下命令安装：

npm install -g openclaw@latest

第二步：获取 APIKey

在查询页面获取您的专属 APIKey：

→ 前往查询 APIKey

第三步：配置 OpenCLAW

1. 运行初始化配置：

openclaw onboard --install-daemon

2. 配置自定义模型（使用 Uantek API）：

openclaw configure

或在配置文件中添加：

// ~/.openclaw/openclaw.json

{

"env": {

"ANTHROPIC_API_KEY": "your-api-key"

},

"agents": {

"defaults": {

"model": {

"primary": "openai/your-model"

}

}

},

"gateway": {

"base_url": "http://nat.ywapi.com:9234/v1"

}

}

第四步：开始使用

配置完成后，可以通过以下方式使用：

  • 在终端输入 openclaw chat 开始对话
  • 打开控制面板 openclaw dashboard
  • 访问 http://localhost:18789 使用 Web 界面

常见问题

Q: 连接失败怎么办？

A: 检查 gateway.base_url 配置是否正确，确保服务器地址可访问。

Q: 没有响应？

A: 确认 APIKey 正确且未过期，检查网络连接。可运行 openclaw doctor 诊断问题。

Q: 其他相似工具？

A: Roo、Cline、Continue 等使用方法一样，配置相同的 API Endpoint 和 API Key 即可。
